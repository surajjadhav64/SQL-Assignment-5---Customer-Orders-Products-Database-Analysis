# SQL-Assignment-5---Customer-Orders-Products-Database-Analysis
## 📌 Overview
This assignment focuses on managing customer, order, and product data using SQL. It demonstrates how relational databases are used to store, retrieve, and analyze business transactions.

## 🚀 Topics Covered
- Database & Table Creation
- Customer Management
- Product Management
- Order Processing
- Data Insertion and Retrieval
- Joins (Customer, Orders, Products)
- Aggregate Functions
- GROUP BY & HAVING
- Filtering and Sorting
- Sales Analysis

## 🎯 Learning Outcomes
- Understanding relational database design
- Managing customer and product information
- Analyzing order and sales data
- Writing efficient SQL queries using joins and aggregations

## 🛠 Technology
- Microsoft SQL Server
